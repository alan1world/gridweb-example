CSS Grid Layout

A powerful layout system that lets you create complex web layouts with ease.

Browser Support

96.23% Global Usage
✅ Modern Browsers

Key Features

  • Two-dimensional layout system
  • Responsive design capabilities
  • Flexible item sizing
  • Grid template areas
  • Automatic placement algorithms
  • Grid line naming
  • Subgrid support (newer browsers)

Basic Grid Example

.container {
  display: grid;
  grid-template-columns: 1fr 2fr 1fr;
  grid-template-rows: auto 1fr auto;
  gap: 1rem;
  min-height: 100vh;
}

.header {
  grid-column: 1 / -1;
  background: #1e293b;
}

.sidebar {
  grid-column: 1 / 2;
  background: #334155;
}

.content {
  grid-column: 2 / 3;
  background: #475569;
}

.footer {
  grid-column: 1 / -1;
  background: #64748b;
}
